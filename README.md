# Placement Preparation Repository

A research-driven, company-wise placement preparation repository built to
maximise the probability of clearing the **Online Assessment (OA)** for the
exact roles applied to. Every document was generated only after researching the
company's actual assessment pattern — nothing here is generic textbook filler.

- **Version:** 1.0
- **Generated:** July 2026
- **Format:** Professionally formatted, study-ready PDFs

## What Each PDF Contains

- **Cover page** with company, applied role(s), version and date.
- **Table of contents** with page numbers.
- **Topic-wise practice questions** with the **correct answer highlighted**.
- **Coding / SQL / Python** documents include the problem statement, input and
  output format, constraints, a worked and **verified** solution, a
  step-by-step explanation, a dry run, and time/space complexity.
- **Mock tests** that mirror the real assessment's section order and mix.

---

## Research Methodology

1. **Role-specific research first.** Each company's OA structure, sections,
   difficulty, coding trends and technical topics were researched before any
   content was written.
2. **Cross-verification.** Findings were checked across multiple sources —
   official pages, candidate interview experiences, community reports and
   trusted preparation platforms — rather than relying on a single source.
3. **Confidence weighting.** Findings were internally classified as HIGH /
   MEDIUM / LOW confidence, and only well-supported patterns drove the PDFs.
4. **Quality over quantity.** Questions reflect real reported patterns and
   company style; low-quality filler and duplicates were avoided.
5. **Verified code.** Every Python solution was executed and every SQL query
   was run in SQLite to confirm it produces the stated output.

---

## How to Use

1. Start with the **topic PDFs** for each section to build coverage.
2. For coding roles, drill the **Coding / Python / SQL** patterns until they
   are second nature.
3. Finish with the **Mock_Test.pdf** for each company under timed conditions to
   simulate the real assessment.
